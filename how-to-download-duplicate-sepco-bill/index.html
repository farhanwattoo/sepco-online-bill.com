How to download duplicate SEPCO bill

To download a duplicate SEPCO bill, open the PITC bill portal with your 14-digit reference number, then use the browser's print dialog and choose "Save as PDF" — the portal has no separate download button, so the print dialog is the download.

This page is only about getting the bill onto your device as a PDF file. For the full background on duplicate bills, start at the duplicate bill pillar guide; for turning the saved file into an acceptable paper copy, see how to print a SEPCO duplicate bill.

Desktop: portal to PDF in four steps

  1. Open the official PITC bill portal via SEPCO's website and enter your 14-digit reference number (recovery tips in where to find it).
  2. When the bill loads, press Ctrl+P (Cmd+P on a Mac) or use the portal's print link to open the print dialog.
  3. In the destination or printer dropdown, choose Save as PDF instead of a physical printer.
  4. Click Save and pick a folder you will remember. Open the file afterwards to confirm the whole bill rendered — occasionally the second page or the payment slip section gets cut, in which case set the print dialog to "Fit to page" and save again.

Chrome on Android

Load the bill in Chrome, tap the three-dot menu, choose Share then Print (or Print directly on some versions), set the printer to Save as PDF, tap the PDF button, and pick a location — Downloads is the default. The file then appears in your Files app and in the Downloads list inside Chrome.

Safari on iPhone

Load the bill in Safari, tap the Share icon, and either choose Print then use the share icon on the print preview to save the PDF, or on newer iOS versions pick Save to Files with the PDF option directly. Save it into a folder in the Files app rather than leaving it in the share sheet, or it can be hard to find later.

Naming and storing the file

  • Rename the file immediately with the month, for example sepco-bill-2026-07.pdf — the portal's default file name tells you nothing.
  • Keep a single folder (Drive, Files, or a WhatsApp "message yourself" chat) and drop each month's PDF there. After a few months you have your own consumption history without depending on the portal keeping old bills visible.
  • Save the PDF as soon as the new bill appears each month. Old months eventually rotate off the portal, and a saved PDF is your only easy copy after that.

Where the downloaded PDF is accepted

A printout of the portal PDF carries the same reference number, billed amount, and due date as the mailed bill, so bank counters and JazzCash/Easypaisa agents accept it for payment, and SEPCO offices accept it as proof of your account details when you raise a complaint. What matters is legibility, not paper quality: the reference number, amount, and due date must be readable. Some formal uses — address proof for other organisations, for instance — may still ask for an office-stamped copy, which your local sub-division office can issue.

No printer at home? You do not need one to download. Save the PDF on your phone now, and take it to any print shop later — the printing guide explains the shop workflow and the settings that keep the copy readable.
